FAQ

FAQ

Is Tailscale required?
Yes, Tailscale is currently required. Tailscale's free plan is more than sufficient for personal use — just install it on both your desktop and mobile and join the same network. Tailscale handles NAT traversal and VPN configuration for you, making secure connections simple.
Where does Claude Code run?
Claude Code runs on your desktop PC. The mobile app only relays terminal input and output. All processing happens on your own machine.
Do I need a public server?
No. RemoCoder operates only within Tailscale's private network. No VPS or cloud server is required.
Does it work on both iPhone and Android?
Yes, both iOS and Android are supported. Built with React Native (Expo), it provides the same experience on both platforms.
How secure is it?
Communication goes through Tailscale's WireGuard encrypted tunnel. UUID token authentication is also required on every connection, preventing unauthorized access even from other devices on the same Tailscale network. No data is sent to any cloud — everything stays between your PC and mobile.
