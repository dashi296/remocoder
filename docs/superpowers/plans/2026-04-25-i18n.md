# docs/index.html 多言語対応（日英）実装プラン

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** RemoCoder ホームページを日英2言語対応にする（`/ja/`・`/en/` URLパス分割、ルートからブラウザ言語で自動リダイレクト）

**Architecture:** CSSを `docs/style.css` に外部化し、`docs/ja/` と `docs/en/` に言語別HTMLを配置。`docs/index.html` は `navigator.language` で振り分けるリダイレクタに置き換える。各ページにhreflangタグと `JA / EN` スイッチャーを追加。

**Tech Stack:** 静的HTML/CSS（ビルドツールなし）

## Task 4: リダイレクタを作成する

**Files:**
- Modify: `docs/index.html`

- [ ] **Step 1: `docs/index.html` をリダイレクタに置き換える**

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <script>
    var lang = navigator.language || navigator.userLanguage || '';
    if (lang.startsWith('ja')) {
      location.replace('./ja/');
    } else {
      location.replace('./en/');
    }
  </script>
  <noscript>
    <meta http-equiv="refresh" content="0; url=./ja/">
  </noscript>
</head>
<body></body>
</html>
```

- [ ] **Step 2: ブラウザで確認する**

- ブラウザの言語設定が日本語の場合: `docs/index.html` を開くと `docs/ja/` にリダイレクトされることを確認
